What is MicroPython

Lean and efficient implementation of the Python3 programming language that is optimized to run on microcontrollers.

MicroPython implements Python 3.4 and some select features of Python 3.5

Code comparison

Blinking a led in pure C

            
            #include <avr/io.h>
            #include <util/delay.h>

            #define BLINK_DELAY_MS 1000

            int main (void)
            {
             DDRB |= _BV(DDB5);

             while(1) {
              PORTB |= _BV(PORTB5);
              _delay_ms(BLINK_DELAY_MS);

              PORTB &= ~_BV(PORTB5);
              _delay_ms(BLINK_DELAY_MS);
             }
            }

Code comparison

Blinking a led in MicroPython

            
                import time
                import machine

                led = machine.Pin(15, machine.Pin.OUT)
                while True:
                    led.high()
                    time.sleep(0.5)
                    led.low()
                    time.sleep(0.5)

How it works

REPL

boot.py and main.py

Main script run at boot time, first it will look for the boot.py and then for the main.py

REPL

MicroPython Interactive Interpreter Mode AKA REPL

(Read-Eval-Print-Loop)

Features:

  • Auto-indent
  • Auto-completion (TAB)
  • Interrupt a running program (Ctrl-C)
  • Paste Mode (Ctrl-E, Ctrl-V, Ctrl-D)
  • Soft Reset (Ctrl-D)

Type of communications

  • Hardware and software SPI
  • I2C
  • OneWire driver
  • WiFi
  • Bluethoot (WIP)
